# 🎓 Graduate Admission Prediction System (Machine Learning)

## 📌 Description
A Machine Learning-based web application designed to predict the chances of admission for graduate programs based on academic performance and profile parameters. The system uses a Linear Regression model trained on real-world data to provide accurate predictions through a user-friendly interface.

## 🚀 Tech Stack
- Programming Language: Python
- Frontend: HTML, CSS
- Backend: Flask
- Machine Learning: Scikit-learn
- Data Analysis: NumPy, Pandas, Matplotlib

## 📊 Dataset
- Source: Kaggle
- The dataset includes features such as GRE score, TOEFL score, CGPA, SOP, LOR, and research experience.

## ⚙️ Features
- Predict admission chances based on user input
- Trained Linear Regression model
- Clean and simple web interface
- Real-time prediction results
- Data preprocessing and visualization

## 🧠 Machine Learning Model
- Algorithm: Linear Regression
- Model trained using Scikit-learn
- Model saved using `.pkl` file for deployment
